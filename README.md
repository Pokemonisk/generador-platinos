# generador-platinos
Español:

Generador de Tarjetas Platino
Una herramienta web sencilla y rápida para crear mosaicos de tus trofeos de PlayStation. Sube tu foto de perfil, tus estadísticas actuales y tus juegos favoritos para generar una tarjeta personalizada lista para compartir.

Características:

*Generación automática de mosaicos en formato imagen.
*Diseño moderno y responsivo.
*Validación de campos para asegurar que toda tu información esté presente.
*Integración de iconos de trofeos.

Inglés:

PlayStation Platinum Card Generator
A fast and simple web tool to create custom mosaics of your PlayStation trophies. Upload your profile picture, enter your stats, and add your favorite games to generate a shareable personalized card.

Features:

*Automatic image mosaic generation.
*Modern and responsive design.
*Field validation to ensure complete information.
*Trophy icon integration.
